Network Analysis

Vehicle Routing Optimization for Emission Reduction

Project Description

This project was completed as part of the “GIS for Climate Action” course offered by Esri. It focuses on optimizing delivery vehicle routes to reduce greenhouse gas emissions, emphasizing the role of GIS-based network analysis in promoting sustainable transportation. By simulating a real-world logistics scenario, I applied advanced routing tools within ArcGIS Pro to identify the most efficient paths that minimize travel time, fuel consumption, and environmental impact.

Data and Tools

  • Customer Order Data: Provided as part of the Esri learning exercise, representing delivery destinations.
  • GIS Software: ArcGIS Pro, used for spatial data visualization, route optimization, and analysis.
  • Network Analysis Tools: Especially the Vehicle Routing Problem (VRP) layer, integral for modeling and solving routing challenges.

Key Highlights

  • Geocoding and Data Preparation: Extracted and prepared customer location points using ArcGIS’s geocoding tool, converting tabular address data into mappable spatial features.
  • Network Dataset and VRP Layer: Constructed a network dataset and added a VRP layer to:
    • Define the central depot (vehicle starting and return point)
    • Assign multiple order points representing delivery locations
    • Specify routing parameters such as time windows, service times, and vehicle capacity
  • Constraint Management: While the analysis encountered limitations due to routing constraints and exceeded network dataset restrictions, the exercise provided critical insight into the complexity of spatial logistics modeling.
  • Output Visualization: Generated map layouts demonstrating proposed delivery routes and vehicle assignments.

Key Learning Outcomes

  • GIS Skill Development: Gained practical experience with ArcGIS Pro's Network Analyst extension, improving my understanding of route optimization and sustainability-focused GIS applications.
  • Climate Action Perspective: Understood how geospatial solutions can directly contribute to climate mitigation efforts by supporting more efficient transportation planning.
  • Problem Solving: Learned how to interpret and troubleshoot analysis limitations, especially those related to routing constraints and network configuration.
