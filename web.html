Project Description

Project Title: Ottawa Neighborhood Web Map (Developed as Part of My Web Mapping Learning Process).

Key Highlights:

  • Tools Used: ArcGIS Online, Instant App.
  • I created an interactive web map in ArcGIS Online by uploading the Ottawa Neighborhood layer and configuring symbology, pop-ups, and attribute visibility to enhance spatial visualization. To make the map more accessible and user-friendly, I also used ArcGIS Instant App to generate a responsive web application.
