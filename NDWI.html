NDWI Analysis of Ottawa

Overview: This project analyzes the Normalized Difference Water Index (NDWI) of Ottawa using satellite imagery in Google Earth Engine.

Methodology

  • Satellite Data: Sentinel-2 imagery
  • Tools Used: Google Earth Engine (GEE), JavaScript
  • NDWI Formula: (Green - NIR) / (Green + NIR)
