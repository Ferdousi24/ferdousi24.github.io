Land Cover Classification Analysis of Bangladesh Using Google Earth Engine

Project Description

This project represents my first hands-on experience with Google Earth Engine (GEE), focusing on analyzing the land cover of Bangladesh. The project was inspired by the GEE tutorial series by Professor Qiusheng Wu, which introduced practical techniques for working with satellite-derived datasets and cloud-based geospatial analysis.

The primary goal of this project was to extract and visualize land cover patterns across Bangladesh using high-resolution Earth observation data. This type of analysis is crucial for understanding land use dynamics, supporting environmental planning, and informing sustainable development initiatives in the region.

Key Highlights

  • Data Source: Utilized the ESA WorldCover 10m 2021 dataset, which provides globally consistent, high-resolution (10m) land cover classification. The dataset was accessed directly through the Google Earth Engine Data Catalog.
  • Google Earth Engine (GEE): Employed GEEâ€™s JavaScript and Python APIs to:
    • Define the study area by filtering the administrative boundaries of Bangladesh
    • Load, visualize, and process raster data representing land cover types
    • Apply masking and color palettes to highlight specific land classes (e.g., urban, cropland, forest, water bodies)
  • Python & Google Colab: Used the geemap and ee libraries in a Google Colab notebook to interact with GEE via Python. This allowed for easy documentation, visualization, and reproducibility of the analysis.
  • Visualization: Generated interactive maps showing spatial distribution of land cover classes, with custom symbology and legends for intuitive interpretation.
  • Skills Gained: Through this project, I gained proficiency in:
    • Accessing and manipulating remote sensing data in Google Earth Engine
    • Integrating GEE with Python for scalable geospatial workflows
    • Using cloud-based environments (Google Colab) for data analysis and sharing

Output Preview

The final output provides a high-resolution map of land cover classifications for Bangladesh in 2021. This forms a foundational step for future projects involving change detection, land use planning, or climate vulnerability assessments.
