Project Description

I processed and visualized squirrel sighting data using ArcGIS Pro to enhance spatial understanding of squirrel distribution across city neighborhoods. My primary objective was to create two distinct map types: a Heat Map to visualize the intensity and concentration of sightings and a Choropleth Map to show the number of squirrel sightings per neighborhood.

  • Data Collection: The squirrel sighting data for this project was collected from iNaturalist, a community-driven platform for wildlife observation, spanning from April 2023 to April 2025. I filtered the data by time to focus on sightings within this specific period. Additionally, the neighborhood boundary data was sourced from Ottawaâ€™s open data platform.

    Key Highlights:

    • 1. Imported the squirrel sighting data from an external CSV source and added it to the Ottawa Neighborhood boundary shapefile in ArcGIS Pro. I made sure to clean the data by removing points located outside the neighborhood boundaries to ensure the accuracy of the spatial analysis.
    • 2. Created a Heat Map to display the intensity of squirrel sightings across different neighborhoods. This map helped visualize areas where sightings were most frequent, indicating potential hotspots of squirrel activity.
    • 3. Used the Spatial Join tool in ArcGIS Pro to associate the squirrel sighting points with the neighborhood polygons. This allowed me to calculate the number of sightings within each neighborhood and create a new field showing the count for each neighborhood.
    • 4. Developed a Choropleth Map to show the total number of squirrel sightings per neighborhood. I applied graduated colors to indicate the density of sightings, making it easy to compare neighborhoods with higher and lower squirrel populations. This map is useful for understanding distribution patterns across the city.

    • Key Learning:

    • Heat Map & Choropleth Map Creation: I learned how to create both heat maps and choropleth maps in ArcGIS Pro to visualize spatial patterns. This allowed me to effectively represent squirrel sighting intensity and distribution across neighborhood.
    • Spatial Join for Data Aggregation: By using the Spatial Join tool, I gained experience in aggregating point data by neighborhood. This process helped me calculate the number of squirrel sightings per neighborhood.
